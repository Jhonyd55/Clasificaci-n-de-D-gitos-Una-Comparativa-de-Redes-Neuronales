Introducción al Proyecto

Este proyecto explora la tarea de clasificación de dígitos escritos a mano utilizando el famoso conjunto de datos MNIST. El objetivo principal es comparar el rendimiento de tres arquitecturas de redes neuronales diferentes: una red neuronal densa, una red neuronal convolucional (CNN) y una red convolucional con técnicas de aumento de datos y Dropout.

A lo largo de este proyecto, se analizan métricas clave como la precisión (accuracy) y la pérdida (loss) para cada modelo. Las visualizaciones a continuación, generadas a partir de los logs de entrenamiento, muestran el progreso de cada modelo a lo largo de las épocas, permitiendo una comparación visual de su desempeño.
